# KDE Setup

## Scripted setup

Run `setup.sh` once after a fresh install. It sets the default terminal to Ghostty and links the necessary desktop/service menu overrides.

## Manual setup

### Audio toggle shortcut

Binds `Meta+Shift+A` to `~/.dotfiles/kde/scripts/toggle-audio.sh`.

System Settings > Keyboard > Shortcuts > Add Command > select the script > set shortcut to `Meta+Shift+A`.
